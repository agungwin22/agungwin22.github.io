Tentang Saya

Saya Agung Cahyono, seorang Freelance Web Developer yang berfokus pada pengembangan aplikasi web menggunakan Python (Flask) serta HTML5, CSS3, dan JavaScript (Vanilla). Saya membantu individu maupun bisnis membangun website dan aplikasi web yang rapi, fungsional, dan mudah dikembangkan.

Saya terbiasa mengerjakan project secara mandiri, berkomunikasi dengan jelas, dan menyelesaikan pekerjaan sesuai kebutuhan klien.
